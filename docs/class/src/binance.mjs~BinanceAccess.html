    public async apiCommand(cmd: object, params: object, accountName: string): * source

    Execute a Binance REST API command. Documentation for the Binance REST API is at https://github.com/binance-exchange/binance-official-api-docs/blob/master/rest-api.md

    Params:

    NameTypeAttributeDescription
    cmd object

    A command definition from @link{BinanceCommands}.
    params object

    An object containing parameters passed to the API call, stored as key,value pairs.
    accountName string

    The name of the Binance account to use for authentication. The account must have been created with loadAccount.

    Return:

    *

    The object returned by the Binance API call (see Binance docs)

    Throw:

    Error

    In the following situations (see error string):

                 - BinanceAccess not initialised,
             - Required parameters missing,
             - Authentication required but no account given,
             - Error returned by Binance
             - Network error

    .

    Example:

      const binance = new BinanceAccess();
  binance.init();
  binance.loadAccount('Zaphod', MyKey, MySecret);
  const info = await binance.apiCommand(
      BinanceCommands.accountInfo,
      {timestamp: this.binance.getTimestamp()},
      'Zaphod'
  );

    public getCoinPair(base: string, quote: string): CoinPair source

    Get a CoinPair object for the given two coins. Using this method is preferable to directly creating a CoinPair as it is safer and avoids duplication.

    Params:

    NameTypeAttributeDescription
    base string

    The code representing the base coin.
    quote string

    The code representing the coin which base is traded against.

    Return:

    CoinPair

    An instance of a CoinPair object.

    Throw:

    Error

    If the coin pair is not available on the exchange.
